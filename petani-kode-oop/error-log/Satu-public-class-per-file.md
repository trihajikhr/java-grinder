# Satu public class per file Java

Ada satu hal yang baru aku tahu juga, adalah bahwa disetiap file `.java`, hanya boleh ada satu class dengan modifier `public`, dan class tersebut harus memiliki nama yang sama dengan nama filenya.

Terserah mau ada berapa banyak class didalam file `.java`, asal yang public hanya ada satu. Jadi class lain yang bukan public di file tersebut, mungkin bisa digunakan sebagai class bantu untuk membantu kinerja class utama yang ber-modifier `public` tersebut
