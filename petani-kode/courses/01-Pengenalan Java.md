## 1 | Apa itu Java
Java adalah bahasa pemrograman yang dibuat oleh seseorang yang bernama James Gosling. 

Bahasa Java adalah bahasa yang secara syntax memiliki banyak kemiripan dengan C/C++, karena memang terinspirasi oleh dua bahasa ini. Java dirancang untuk tujuan umum, dan sepenuhnya menganut paradigma OOP. Oleh karena itulah, setiap membuat program Java, maka wajib hukumnya membuat objek terlebih dahulu.

Tapi untuk pembelajaran, konsep OOP tidak akan terlalu ditekankan, jadi masih sedikit menggunakan konsep prosedural.

## 2 | Sejarah singkat Java
Awalmnya Java bernama _oak_, lalu dirubah menjadi _Green_, lalu diubah lagi menjadi _Java_, yang mana nama Java sebenarnya muncul karena terinspirasi oleh asal usul kopi yang diminum Gosling, yaitu Pulau Jawa. Itulah alasan kenapa logo Java berbentuk secangkir kopi. 

## 3 | JDK, J2SE, Java SE, dan Java EE
JDK adalah singkatan dari Java Development Kit, berisi tools, library, dan API yang diperlukan untuk membuat program dengan Java. Termasuk didalamnya terdapat JVM.

Java JDK memiliki beberapa edisi, yaitu Java SE (_Standarrt Edition_), Java EE (_Enterprise Edition_), dan Java ME (_Micro Edition_).

Java SE atau J2SE adalah Java yang digunakan untuk membuat aplikasi standar seperti aplikasi desktop dan command line (CLI).

Sedangkan Java EE adalah edisi enterprise yang digunakan untuk membuat aplikasi enterprise seperti dalam Bank, Industri, dll. Namun, Java EE sekarang sudah berubah nama menjadi Jakarta EE.

Lalu, Java ME adalah Java versi mungil yang digunakan untuk membuat aplikasi mobile, micro controller, sensor, dll.

## 4 | MEngapa Belajar Java
1. **Java bersifat General Purpose**, atau tidak hanya dipakai untuk satu program spesifik saja, melainkan di banyak sistem. Ini seperti kunci inggris yang bisa digunakan untuk membuka banyak ukuran baut dan mur sekaligus. Oleh karena itu, Java bisa digunakan untuk aplikasi desktop, android, dan juga web.

2. **Cross Platform**, artinya bisa berjalan di platform yang berbeda-beda, seperti Windows, MacOS, dan Linux. Semisal di windows  Dengan begini, kita tidak perlu pusing untuk membuat binary untuk tiap-tiap platform.Misalnya, untuk windows harus digunakan `.exe`, Linux `.deb` atau `.rpm`, dan Mac `.dmg`. Tetapi cukup dengan file `jar` saja, jika di platform tersebut sudah terinstal Java, maka program tersebut akan langsung bisa dibuka.
3. **Membuat Aplikasi Android**, walaupun sekarang Google lebih mengandalkan Kotlin untuk membuat aplikasi Android, namun tidak ada salahnya untuk belajar Java terlebih dahulu. Kotlin memang lebih singkat dibandingkan Java, namun dibalik kode yang singkat itu, ada peran Java. Memahami dan menguasai Java akan membuat proses belajar Kotlin jauh lebih mudah.
4. **Java merupakan bahasa populer**, kenapa ini penting? Karena semakin populer suatu bahasa, maka semakin ramai dan besar komunitasnya, artinya ada banyak ruang untuk berdiskusi dan bertanya.
